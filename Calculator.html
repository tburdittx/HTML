Calculator

Building a calculator is a good example of seeing HTML, CSS and Javascript all working together to create something on the webpage.

The HTML is used to structure the calculator and decide where everything needs to go. This is done using the tabling function. The design of the calculator is done using CSS, and this is held in a seperate piece of code. Finally, the JavaScript is used to bring the calculator to life, adding in the functionality and methods a calculator would need.
